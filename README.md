Finance-Tracker-Insights-Dashboard

📌 Description
This project presents an interactive Personal Finance Dashboard built using Power BI to analyze income, expenses, and savings patterns. The dashboard provides actionable insights to improve financial decision-making.


🎯 Objectives
Track income, expenses, and savings
Analyze spending patterns
Monitor financial growth
Identify high-expense categories


📊 Features
KPI Cards (Income, Expense, Savings)
Expense category analysis (Bar chart)
Trend analysis (Line chart)
